Simulink GN&C Mini-Stack

In progress Simulink Attitude Control Monte Carlo-ready

A 3D Simulink simulation of a small-satellite reaction wheel attitude control system (6-DOF), including sensor models (IMU + star sensor) with noise, onboard signal processing, and closed-loop control.

Overview

Goal

Demonstrate a GN&C-relevant Simulink workflow: plant dynamics + sensors + estimation/signal processing + controller + performance verification.

System

  • 6-DOF small-sat attitude dynamics in orbit (3D visualization-driven workflow)
  • Reaction wheel actuator model with saturation / rate limits (as applicable)
  • Disturbance & noise injection for robustness testing

Sensors

  • IMU model (gyro/accel behavior as implemented) with configurable noise
  • Star sensor measurement model with noise and update-rate effects
  • Signal conditioning / processing blocks to emulate onboard workflows

What’s implemented so far

Core loop

  • Closed-loop attitude control with reaction wheel actuation
  • Sensor noise injection (IMU + star sensor)
  • Onboard signal processing blocks feeding the controller

Verification (initial)

  • Baseline tracking performance runs
  • Noise sensitivity runs (sensor noise sweep)
  • Disturbance rejection runs (representative disturbance cases)

How I’m validating it

This project is intentionally built to support verification and robustness evaluation: repeatable scenarios, parameterized noise/disturbances, and clear performance metrics.
